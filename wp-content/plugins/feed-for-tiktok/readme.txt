Displays the feed of any user on TikTok plus account information. Available for Elementor and shortcode.

== Description ==

TikTok has become one of the most powerful social media platforms. For this reason today you can enjoy a super plugin through which you can display the feed of your TikTok account directly on the website. This plugin has a 100% customizable version for Elementor.

The feed will be displayed in carousel format.

This plugin offers you the possibility to display the general information of your account such as the number of followers, the number of people you follow, the number of likes, the number of videos, etc.

== Installation ==

1. Go to the Plugins Menu in WordPress
2. Search for "Feed for TikTok"
3. Click "Install"
4. Go to Tik Tok in your admin dashboard
5. Click the Connect to the API button
6. Configure your TikTok feed
7. Press "Save" button and then copy & paste your shortcode in any page or post or use elementor widgets
